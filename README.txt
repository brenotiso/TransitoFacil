---------------

COMO O ROGRAMA FUNCIONA:

O programa consiste em uma tela principal onde há quatro botões.

Cada um dos botões possui uma descrição que irá aparecer quando passar o mouse sobre o mesmo.

O botão Placas redireciona o usuário para um nova tela onde ele deverá escolher uma categoria
de placas, após a escolha o usuário poderá ver cada uma das placas da respectiva categoria.

O botão Trânsito geral leva o utilizador para uma tela contendo informações importantes sobre
conhecimentos no trânsito.

O botão Simulado de placas inicia um simulado sobre o conhecimento das diversas placas, é importante
notar que o botão de terminar somente aparecerá quando todas as perguntas estiverem respondidas,
assim será exibido um gráfico mostrando os acertos e erros do usuário. As questões do simulado são 
escolhidas de forma aleatória.

O botão Simulado de Trânsito geral funciona de forma análoga ao botão anterior, porém neste as perguntas
não são limitadas a placas e todas, também, deverão ser respondidas.

No final da tela principal existe ainda uma opção, escrita em azul, para adicionar novas questões, ao clicar
em "Adicionar questões" o usuário deverá escolher um arquivo .txt que tenha a mesma sintaxe do arquivo EXEMPLO.txt
que acompanha este projeto. Os arquivos entradaGeral.txt e entradaPlacas.txt mostram questões aptas a serem adicionadas.
